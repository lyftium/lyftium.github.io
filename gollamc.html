Open Source Aerospace CAD Framework

GOLLAMC: Vision-to-CAD for Aerospace Engineering

A framework combining Jack Dorsey's Goose Agents, Ollama, and MCP servers for CAD generation. Interact with technical documents through natural language and generate 3D models and 2D drawings with 100% local execution.

How GOLLAMC Works

1

Input Technical Documents

Upload PDFs, images, or technical specifications. GOLLAMC processes these documents using vision capabilities to extract engineering data.

2

Natural Language Interaction

Interact with your documents using natural language. Ask questions, request modifications, or specify CAD requirements through conversational AI.

3

Multi-Agent Processing

Goose Agents coordinate with Claude Code to analyze requirements, generate specifications, and create CAD models through intelligent agent handoffs.

4

CAD Generation

Generate 3D models and 2D drawings based on extracted specifications. Output ready for aerospace engineering workflows with proper tolerances and standards.

Technical Architecture

Goose Agents - Jack Dorsey's agent framework for task orchestration
Ollama - Local LLM inference engine for privacy-focused AI
MCP Servers - Model Context Protocol for tool integration
Claude Code - AI-powered coding assistant for CAD generation
Smarteam PLM - Optional PLM integration for enterprise workflows
